### Deskripsi Project
Project ini merupakan aplikasi manajemen klinik sederhana berbasis Python yang dibuat
untuk memenuhi Ujian Akhir Semester mata kuliah Pemrograman Berorientasi Objek (PBO).
Aplikasi ini digunakan untuk mengelola data pasien, dokter, pendaftaran berobat,
dan rekam medis dengan menerapkan konsep OOP.

---

### Konsep OOP yang Digunakan
- Class dan Object  
- Encapsulation  
- Modular Programming  
- Relasi Antar Objek  

---

### Fitur Aplikasi
- Menampilkan data pasien  
- Menampilkan data dokter  
- Pendaftaran pasien berobat  
- Penyimpanan dan tampilan rekam medis
